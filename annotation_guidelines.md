# Deliverable 5 – Human Evaluation of Slovak Language Model Output

## 1. Objective

The purpose of this human evaluation is to assess the quality of text generated by a language model trained on Slovak. Since automatic metrics do not fully capture linguistic quality in morphologically rich languages, structured human judgments are used.


## 2. Evaluation Setup

* **Annotators**: 3 students, one of which a native Slovak speaker
* **Material**: Model-generated Slovak texts evaluated independently
* **Method**: Likert-scale ratings per category
* **Reliability**: Inter-annotator agreement measured using Krippendorff’s α


## 3. Evaluation Categories and Guidelines

### Category 1: Syntactic Fluency

**Definition**
Measures grammatical correctness and fluency, including morphology, word order, agreement, and punctuation.

**Scale (1–5)**

| Score | Description                      |
| ----- | -------------------------------- |
| 1     | Severely ungrammatical           |
| 2     | Many grammatical errors          |
| 3     | Mostly grammatical, minor errors |
| 4     | Grammatically correct            |
| 5     | Fully fluent and error-free      |

**Examples**

* **1**: *„Ja včera ísť obchod kúpiť.“*
* **3**: *„Včera som išiel do obchodu kúpiť jedla.“*
* **5**: *„Včera som išiel do obchodu kúpiť jedlo.“*

### Category 2: Semantic Coherence

**Definition**
Assesses whether the text is meaningful, logically consistent, and coherent as a whole.

**Scale (1–5)**

| Score | Description                    |
| ----- | ------------------------------ |
| 1     | Meaningless or contradictory   |
| 2     | Largely incoherent             |
| 3     | Generally coherent with issues |
| 4     | Coherent and understandable    |
| 5     | Fully coherent and clear       |

**Examples**

* **1**: *„Slovensko je strom, ktorý rozmýšľa.“*
* **3**: *„Slovensko je krajina v Európe a má veľa miest.“*
* **5**: *„Slovensko je krajina v strednej Európe s bohatou históriou.“*

## 4. Aggregation and Reporting

Scores are averaged across annotators and samples. Category-wise results are reported with confidence intervals, and qualitative error analysis is used to interpret systematic weaknesses.
