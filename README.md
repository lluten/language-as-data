# language-as-data
This repository contains all code and data used for the seminar Language as Data.

### 1. Clone the repository
```bash
git clone https://github.com/lluten/language-as-data.git
cd langauge-as-data
```
### 2. (Optional) Create and activate a virtual environment
```bash
python -m venv .venv
source .venv/bin/activate
```
### 3. Install dependencies and create folder structure
```bash
chmod +x setup.sh
./setup.sh
```

### 4. Download data
```bash
chmod +x download_data.sh
./download_data.sh
```
